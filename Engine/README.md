# Additive Engine

3 Sequence selectors (A,B & C) combined with 2 sequence operators (OP1, OP2)

A and B are combined using Operator 1, the result of this is combined with C using operator 2

Selected Partials = (A OP1 B) OP2 C

e.g. (A AND B) XOR C
